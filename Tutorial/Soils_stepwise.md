# Soils.mp4 Step-by-Step Notes

This write-up is based on the visible on-screen actions in `Soils.mp4`.

## Workflow Summary

1. Open the USDA **Web Soil Survey** site.
2. Stay on the **Area of Interest (AOI)** tab.
3. In the left panel, expand **Import AOI**.
4. Open **Create AOI from Shapefile**.
5. Use the file-picker windows to locate the project boundary shapefile components.
6. Select the matching shapefile files for the AOI import:
   - `.shp`
   - `.shx`
   - `.prj`
   - `.dbf` (optional, but shown in the video)
7. The files shown in the recording appear to be named similar to `RRG-2 Project Boundary`.
8. If the site shows prefix-mismatch warnings, correct the file selections so the shapefile parts use the same base name.
9. Click **Set AOI** to import the shapefile boundary.
10. Wait for the AOI to load on the map.
11. Confirm the AOI appears as a hatched polygon over the imagery.
12. In the AOI properties panel, confirm the imported AOI name appears as **RRG-2 Project Boundary**.
13. Confirm Web Soil Survey data is available for **Catahoula Parish, Louisiana (LA025)**.
14. Note the area shown in the recording is about **49.7 acres**.

## Soil Map Review

15. Open the **Soil Map** tab.
16. Wait for the map and legend to load.
17. Review the map unit legend for the AOI.
18. The visible map units in the recording are:
   - `AA`
   - `Gy`
   - `SW`
19. Review the acreage and percent-of-AOI values shown in the legend table.

## Soil Data Explorer Review

20. Open the **Soil Data Explorer** tab.
21. Open the **Soil Reports** section.
22. Expand **Vegetative Productivity**.
23. Select **Forestland Productivity**.
24. Click **View Soil Report**.
25. Review the forestland productivity output for the AOI.

## Suitabilities And Limitations Review

26. Switch to **Suitabilities and Limitations for Use**.
27. Expand **Land Management**.
28. Select **Erosion Hazard (Off-Road, Off-Trail)**.
29. Click **View Rating**.
30. Review the map display for the erosion-hazard rating inside the AOI.
31. In the left options panel, enable or confirm the displayed options:
   - **Map**
   - **Table**
   - **Component Breakdown and Rating Reasons**
   - **Numeric Values**
   - **Description of Rating**
   - **Detailed Description**
32. Click **View Description** to open the detailed interpretation text.
33. Read the explanatory popup for **Erosion Hazard (Off-Road, Off-Trail)**.

## Main Outcome Shown In The Video

34. The overall process shown is:
   - import a project boundary shapefile as an AOI,
   - verify the AOI and soil map units,
   - open Soil Data Explorer outputs,
   - review forestland productivity,
   - and review erosion-hazard suitability information for the selected property.
